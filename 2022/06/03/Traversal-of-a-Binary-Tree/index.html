Traversal of a Binary Tree

Intro

There are two ways of traversal of a binary tree, which is depth traversal and breadth traversal. Depth traversal includes three main approaches: Pre-order traversal, Middle-order traversal, and Post-order traversal. Breadth traversal includes Hierarchical traversal which is traversing down one level at a time.

Example

Pre-order traversal

Root node -> Left branch -> Right branch
The sequence of the above example would be: 8, 3, 1, 6, 4, 7, 10, 14, 13

Middle-order traversal

Left branch -> Root node -> Right branch
The sequence of the above example would be: 1, 3, 4, 6, 7, 8, 10, 13, 14

Post-order traversal

Left branch -> Right branch -> Root node
The sequence of the above example would be: 1, 4, 7, 6, 3, 13, 14, 10, 8

Hierarchical traversal

The sequence of the above example would be: 8, 3, 10, 1, 6, 14, 4, 7, 13
